# Release v0.23.1

> **Release Date**: 2026-02-04
> **Upgrade Type**: Non-breaking (rolling upgrade)
> **Upgrade Height**: N/A - No coordination required

## Overview

Security patch release that bumps CometBFT to v0.38.21 to address a critical vulnerability disclosed in CSA-2026-001.

## Upgrade Instructions

### For Validators

This is a **non-breaking security patch**. No coordinated upgrade is required.

```bash
# Using cosmovisor (recommended)
# Stop node, replace binary, restart

# Manual upgrade
cd go-bitsong
git fetch --tags
git checkout v0.23.1
make install
# Restart your node
```

### Verification

```bash
# Verify version
bitsongd version
# Expected: 0.23.1

# Verify CometBFT version
bitsongd version --long | grep cometbft
# Expected: v0.38.21
```

## Changes

### Security

- **CometBFT**: Bump to v0.38.21 ([CSA-2026-001](https://github.com/bitsongofficial/internal-docs/blob/main/docs/operations/security/advisories/CSA-2026-001.md))

### CI

- Modernize release workflow with multi-arch builds and checksums
- Add changelog enforcement and conventional commit validation
- Fix shell quoting in GITHUB_OUTPUT writes
